HOW TO USE
================================================================================

1. INSTALL DEPENDENCIES:
   pip install -r requirements.txt

2. CREATE .env FILE:
   FRAGELLA_API_KEY=your_api_key_here

3. RUN APPLICATION:
   streamlit run scentify.py

4. USE ML FEATURES:
   a) Add 2+ perfumes to inventory
   b) Click "Get AI Recommendations" on home page
   c) Configure settings (optional)
   d) Click "Generate AI Recommendations"
   e) View personalized results

5. RUN TESTS:
   python test_ml.py

CONFIGURATION
================================================================================

Location: scentify.py, ML_CONFIG dictionary

Settings:
  - negative_samples_ratio: 2 (negatives per positive)
  - min_inventory_size: 2 (minimum perfumes for ML)
  - model_type: 'logistic_regression' (default algorithm)
  - random_state: 42 (reproducibility)
  - min_recommendation_probability: 0.5 (threshold)
  - diversity_threshold: 0.3 (diversity level)

All settings adjustable via UI or code.
